# AnimeGamesProxy - 动漫游戏安卓代理模块

## 简介
AnimeGamesProxy 是一个面向安卓平台的动漫手游代理模块，用于将游戏网络请求重定向至指定私服

---

## 支持游戏列表

### 开放空间（TapTap）
- 启用GM

### 碧蓝档案（日服 & 国际服）
- 禁用反作弊

### 星塔旅人
- 移除Google相关依赖

### 明日方舟（仅国服）
- 禁用 `NetWorkConfig` 校验

### 鸣潮
- 缺少测试

### 原神
- 支持 RSA Patch
- 适用于 3.1 至最新版本
- 支持 Beta 测试版本修补

### 绝区零
- 缺少测试

### 崩坏：星穹铁道

### 崩坏3
- 缺少测试

## 静默模式

通过向 APK 的 `assets` 目录中添加有效的 `agp_config.xml` 配置文件，并使用 LSPatch 修补 APK，可启用静默模式

启用后：

- 不会弹出设置窗口
- 模块将始终从 APK 读取配置
- 适用于固定服务器连接场景

你可以先在游戏内完成配置，然后通过：

```text
菜单 → 配置管理 → 导出配置
```

将当前配置导出到Download

---

## 注意事项

- 本模块不支持在 PC 模拟器中运行
- 在模拟器环境下，部分功能可能失效或无法正常工作

---

## 无 Root 设备使用方式

如果设备没有 Root 权限，可配合以下框架使用：

- [LSPatch](https://github.com/JingMatrix/LSPatch)
- [NPatch（推荐）](https://github.com/7723mod/NPatch)
